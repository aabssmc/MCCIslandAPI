## Installation
Gradle:
```gradle
repositories {
  mavenCentral()
  maven { url = "https://jitpack.io" }
}

dependencies {
  implementation("cc.aabss:MCCIslandAPI:1.0")
}
```

Maven:
```xml
  <repositories>
    <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
    </repository>
  </repositories>
  <dependency>
    <groupId>cc.aabss</groupId>
    <artifactId>MCCIslandAPI</artifactId>
    <version>1.0</version>
  </dependency>
```
